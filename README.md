# Opentelemetry with Opensearch backend
This is a sample docker compose setup with:
* [jaeger-hotrod](https://github.com/jaegertracing/jaeger/tree/master/examples/hotrod)
* [opentelemtery agent](https://opentelemetry.io/docs/collector/getting-started/)
* [opentelemtery collector](https://opentelemetry.io/docs/collector/getting-started/)
* [opensearch data-pepper](https://opensearch.org/docs/latest/observability-plugins/trace/data-prepper/)
* [opensearch node](https://opensearch.org/)
* [opensearch-dashboards](https://opensearch.org/docs/latest/dashboards/index/#get-started-with-opensearch-dashboards)
